### 💡 Part 1 Key Pearls
- 急性腹痛要找"fat stranding"的常見原因：急性胰臟炎、急性膽囊炎
- 壞死性胰臟炎延遲併發：千萬不要忘記 Pseudoaneurysm（假性动脉瘤）！
- 膽囊炎的進程：Simple → Gangrenous → Perforated
- Gallstone Ileus = Rigler Triad（膽石、腸阻塞、膽道積氣）

### 💡 Part 2 Key Pearls
- **RLQ Pain 鑑別診斷**：Appendicitis / Cecal Diverticulitis / Epiploic Appendagitis / Omental Infarct / Crohn's / Ovarian Torsion
- Epiploic Appendagitis 常被誤診為 Diverticulitis，重點是脂肪密度病變
- **Ovarian Torsion** = 婦科急症！超聲看到「目標徵候」
- Crohn's 三型：炎症性（inflammatory）→ 狹窄性（stricturing）→ 穿透性（penetrating）
- **TOA** = Tubo-Ovarian Abscess，常見於 PID 併發

### 💡 Part 3 Key Pearls
- **Closed loop obstruction** = 外科急症！CT 可见「鳥嘴徵」（bird beak sign）
- **Sigmoid Volvulus** = 乙狀結腸扭轉，「咖啡豆徵」（coffee bean sign）
- **Ischemic Bowel** = 門靜脈積氣（portal venous gas）＋腸壁積氣（pneumatosis intestinalis）
- **Typhlitis** = 常見於化療後嗜中性球減少病患
- **C diff colitis** = 偽膜性結腸炎，見於抗生素治療後
- **Meckel's Diverticulitis** = 永遠要記住的「法則」：2%人口、2呎逆迴腸、2種組織（胃黏膜）

## 🔵 Part 4：Misc Challenging Cases（其他困難案例）

| 48 | **Ovarian Vein Thrombosis**（卵巢靜脈血栓）⚠️ | Challenging |
| 49 | **SBO caused by mesodiverticular band**（中腸系膜憩室帶造成小腸阻塞） | Moderate |
| 50 | **IgG4 disease**（免疫球蛋白G4相關疾病）⚠️ | Moderate |
| 51 | **DIOS (Distal Intestinal Obstruction Syndrome)**（遠端腸道阻塞症候群） | Moderate |
| 52 | **Groove Pancreatitis**（胰溝部胰臟炎）⚠️ | Challenging |

### 💡 Part 4 Key Pearls
- **Gallbladder Volvulus** = 膽囊游離造成扭轉，CT 可見「游離膽囊徵」
- **Ovarian Vein Thrombosis** = 產後發燒的重要鑑別，需抗凝治療
- **IgG4 disease** = 自身免疫性胰臟炎（AIP），可模擬胰臟癌
- **Groove Pancreatitis** = 發生在胰頭與十二指腸溝部，常見於酗酒男性
